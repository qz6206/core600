# Core 600 品牌资源

## 配色

| 颜色 | 色值 | 用途 |
|------|------|------|
| 红色 | `#dc2626` | 外框 C |
| 黑色 | `#0f172a` | 内 L 折线（数字 6 的下半部分） |
| 白色 | `#ffffff` | 深色背景下的 L 折线 |

## SVG 矢量文件（推荐使用，无限缩放）

| 文件 | 用途 |
|------|------|
| `logo.svg` | 主 Logo（图标 + 文字，浅色背景）|
| `logo-dark.svg` | 主 Logo 深色背景版（文字白色） |
| `icon.svg` | 纯图标（适合小尺寸场景）|
| `icon-square.svg` | App 图标（200x200，圆角方形）|
| `icon-circle.svg` | 头像版（圆形，适合微信/X 等）|
| `social-banner.svg` | 社交分享卡片（1200x630）|

## PNG 位图文件（直接使用）

### 纯图标
- `icon-32.png` / `icon-64.png` / `icon-128.png` / `icon-256.png` / `icon-512.png`

### App 图标（圆角方形）
- `app-icon-200.png` / `app-icon-512.png` / `app-icon-1024.png`（提交 App Store 用）

### 头像（圆形）
- `avatar-200.png` — 微信头像、X/Twitter 头像
- `avatar-400.png` — 高清头像

### 横版 Logo
- `logo-60.png` / `logo-120.png` / `logo-240.png` — 浅色背景
- `logo-dark-60.png` / `logo-dark-120.png` — 深色背景

### 社交分享
- `social-banner.png` — Twitter/Facebook/微信链接预览图

## 使用建议

| 场景 | 推荐文件 |
|------|---------|
| 网站头部 | `logo.svg` |
| 浏览器 favicon | `icon-32.png` |
| App 图标 | `app-icon-1024.png`（iOS 提交） |
| 微信头像 | `avatar-400.png` |
| X / Twitter 头像 | `avatar-400.png` |
| 名片印刷 | `logo.svg`（矢量，无限放大）|
| PPT / Keynote | `logo.svg` |
| 社交分享缩略图 | `social-banner.png` |
| 衣服/马克杯印花 | `icon.svg`（矢量，可任意放大） |

## 编辑

SVG 文件可在以下软件打开编辑：
- **Figma** —— 免费在线设计工具，最推荐
- **Adobe Illustrator** —— 专业级矢量设计
- **Inkscape** —— 免费开源
- **Sketch** —— Mac 专属

PNG 文件可在 Photoshop、GIMP、Figma 等中编辑。
